## User Story 10: Confirm Delivery

**Title**: Confirm Delivery

**Description**: As a coffee delivery robot, I want to confirm the delivery with the customer so that I can ensure the order is completed successfully.

**Priority**: High

**Risk**: Medium

**Estimate**: 2 days

**Acceptance Criteria**:
- The robot should prompt the customer to confirm delivery via voice command.
- The robot should update the order status to "delivered" upon confirmation.
- The robot should have a mechanism to retry confirmation if the first attempt fails.
